# 🍅 LLM Timer - 作品简介

## 📋 作品简介

**LLM Timer** 是一款结合番茄工作法、任务管理和AI智能排期的个人时间管理系统。通过集成大语言模型（DeepSeek API），系统能够理解自然语言输入，自动为任务安排合理的时间段，并实时追踪任务完成进度。无论是学生、职场人士还是自由职业者，都能通过这款工具提升时间管理效率，培养专注习惯，实现更高效的工作与学习。

### 核心亮点

- **🤖 AI驱动的智能排期**：基于大语言模型，理解自然语言输入，自动识别时间、优先级等关键信息
- **⏱️ 自适应番茄钟**：专注时长自动匹配任务需求，告别固定的25分钟限制
- **📊 连续进度追踪**：基于实际完成分钟数的连续进度显示，精准反映任务完成情况
- **📅 可视化日历排期**：FullCalendar集成，直观查看和管理任务时间安排
- **💾 本地数据存储**：所有数据存储在本地，保护隐私安全

## 🔧 解决的问题

### 1. **任务排期复杂且耗时**
   - **问题**：手动安排任务时间需要考虑优先级、时长、现有安排等多个因素，容易遗漏或产生冲突
   - **解决方案**：AI智能排期功能自动分析任务特征，综合考虑优先级、时长、现有安排，自动分配最优时间段，避免时间冲突
   - **效果**：排期时间从数分钟缩短到秒级，准确率显著提升

### 2. **自然语言输入不友好**
   - **问题**：传统任务管理工具需要填写多个表单字段，输入繁琐，不符合自然表达习惯
   - **解决方案**：支持自然语言输入，如"明天上午安排阅读1小时"，系统自动解析出任务标题、时间、时长等信息
   - **效果**：输入效率提升3-5倍，用户体验更加流畅

### 3. **番茄钟时长固定不灵活**
   - **问题**：传统番茄工作法固定25分钟专注时长，不适合所有任务类型，短任务浪费休息时间，长任务需要多次拆分
   - **解决方案**：专注时长自动设置为任务的预估时长，根据任务实际需求灵活调整
   - **效果**：专注时长匹配度提升，避免不必要的中断，提升专注体验

### 4. **进度追踪不精确**
   - **问题**：传统番茄钟进度显示为"0/3"这样的间断计数，无法准确反映实际完成情况（如完成了25分钟中的20分钟）
   - **解决方案**：基于实际完成分钟数的连续进度显示，实时精确反映任务完成百分比
   - **效果**：进度可视化更准确，用户能清楚了解任务完成情况

### 5. **时间表达识别困难**
   - **问题**：用户说"明天"、"上午"等相对时间表达，系统无法准确理解并转换为具体日期时间
   - **解决方案**：AI自动识别相对时间表达，结合当前日期时间，准确转换为具体的时间点
   - **效果**：时间解析准确率接近100%，支持多种中文时间表达方式

### 6. **数据分散难以管理**
   - **问题**：任务列表、日历、计时器分散在不同界面，需要频繁切换才能了解全貌
   - **解决方案**：集成任务列表、日历视图、番茄钟于一体，统一管理，实时同步
   - **效果**：减少界面切换，提升操作效率，数据一致性得到保证

### 7. **缺乏智能决策支持**
   - **问题**：用户需要自己判断任务优先级、安排顺序，容易受主观因素影响
   - **解决方案**：AI根据任务描述自动判断优先级，智能排期时考虑优先级和时长因素
   - **效果**：排期决策更科学，重要任务优先处理，提升整体效率
